# Oracle Assignment

> Frontend service for Oracle Assignment

## Prerequisites

```
Node.js  >=12.16.3
Yarn     >=1.19.0
```

## Install dependencies

```bash
yarn install
```

## Build

```
# Development build
yarn dev

# Production build
yarn build
```

## Run unit tests

```
yarn test
```

## Lint and fix files

```
yarn lint
```
